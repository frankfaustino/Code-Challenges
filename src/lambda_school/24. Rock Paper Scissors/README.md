# Rock Paper Scissors

Write a function called `rockPaperScissors` that will take a number `n` and output all the possibilities.

**Example:**<br />

```js
rockPaperScissors(1)
/* [[rock], [paper], [scissors]] */

rockPaperScissors(2)
/*
  [
    [rock, rock],
    [rock, paper],
    [rock, scissors],
    [paper, rock],
    [paper, paper],
    [paper, scissors],
    [scissors, rock],
    [scissors, paper],
    [scissors, scissors]
  ]
*/
```
